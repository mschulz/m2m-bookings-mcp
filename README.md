# m2m-booking-system

This server is to collect all the data on bookings from Launch27 via Zapier.

To start using the template, first generate a new API_KEY:
```
python generate_key.py
```
Insert this key into config.py as API_KEY.

To set up the configuration, set the environment variable. Typically:
```
export APP_SETTINGS=config.DevelopmentConfig
```
See config.py for the other possible configuartions.
